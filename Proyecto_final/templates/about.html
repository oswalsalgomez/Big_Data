BigData-MiProyecto

Descripción del Proyecto

Este proyecto es una aplicación web desarrollada para la gestión y el análisis de las resoluciones emitidas por la Autoridad Nacional de Licencias Ambientales utilizando MongoDB y Elasticsearch. Su objetivo es analizar a que se refiere cada una de las resoluciones con el fin de determinar orientaciones para dar respuesta a requerimientos de esa autoridad mediante la análitica de datos.
